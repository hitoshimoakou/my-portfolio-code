# 雪かき依頼フォームアプリ（yukikaki-app）

## 概要
地域の人が「雪かき依頼」を気軽に送れるフォームを作り、その内容をGoogleスプレッドシートに保存するWebアプリです。

## 使用技術
- Google Apps Script
- Google Forms / Google Sheets

## 機能
- フォーム入力→スプレッドシート自動記録
- 利用者からの依頼を自動分類・通知（例：ChatworkやSlack連携可能）

## 今後の展望
- 通知先を選べる機能追加
- 簡易なステータス管理（対応済/未対応）
